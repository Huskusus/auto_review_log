# auto_review_log
Bash-скрипт для автоматического анализа лог-файлов на наличие ошибок.      Прием на вход адрес директории с логами;     Поиск по множеству паттернов (error, failed, critical);     Подсчет общего количества ошибок;     Вывод последних ошибок;     Сохранение отчета в формате .txt в корневой папке /home.
